# Guess Game

"Угадай число", где компьютер загадывает число от 1 до 100, а игрок пытается его угадать. Программа использует алгоритм бинарного поиска для подсказок и включает библиотеку для работы с римскими цифрами.

## Установка

```bash
# Сборка
autoreconf -fisv
./configure
make

# Установка
sudo make install

# Запуск
guess
guess -r

# Справка
man guess
guess --help

# Документация
/usr/local/share/doc/guess/html/

# Тестирование
make check

# Деинсталляция
sudo make uninstall
make clean
                                                                                                                            
```
